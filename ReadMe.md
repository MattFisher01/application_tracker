# App Tracker

A simple app to track job applications

## Description

The app uses a tabular front end to display upcoming applications you are currently working on and those you have made.
The app allows you to set the status of each application, add notes and track a contacts information, whether that be a company email address of a third party recruiter (as well as their recruitment company info)


## Getting started
* pip install requirements.txt, preferably into a virtual env
* run the app using `flask --app main.py run`
